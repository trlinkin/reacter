Integrates the output from your monitoring systems so you can decide what to do to it, where to put it, and when.

Getting Started

Install

Install Reacter Core from the Python Package Index (PyPI), and checkout a copy of the reacter-agents repository. You can also install Reacter using Puppet, or directly from the .deb or .rpm

Configure

Configure Reacter to handle incoming metrics. Reacter's agents can manipulate the incoming messages, format them, route them to another monitoring system, log them to a file, or prepare them for other agents to consume.

Run

Start the reacter service and begin submitting metrics to your message queue or other intermediary data store.
