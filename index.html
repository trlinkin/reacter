Getting Started

Install

Install Reacter from RubyGems (gem install reacter). You can also install Reacter using Puppet, Chef, or directly from the .deb or .rpm packages.

Configure

Configure Reacter to handle incoming metrics. Reacter's agents can manipulate the incoming messages, format them, log them to a file, or relay them to another instance of Reacter.

Run

Start reacter and begin submitting metrics. When Reacter receives them, the data will be swiftly handled according to your exacting and detailed requirements.
